## Notation

| Symbol | Meaning |
|--------|---------|
| $(\Omega, \mathcal{F}, \mathbb{P})$ | A probability space, where $\Omega$ is the sample space, $\mathcal{F}$ is a $\sigma$-field on $\Omega$, and $\mathbb{P}$ is a probability measure on $\mathcal{F}$ |
| $\mathcal{P}(\Omega)$ | The power set of $\Omega$ |
| $A^c$ | The complement of $A$ in $\Omega$ |
| $\sigma(\mathcal{G})$ | The $\sigma$-field generated by $\mathcal{G}$ |
| $\mathcal{B}(\mathbb{R})$ | The Borel $\sigma$-field on $\mathbb{R}$ |
| $A_n \uparrow A$ | $A_n \subseteq A_{n+1}$ for all $n$ and $A = \bigcup_{n=1}^\infty A_n$ |
| $A_n \downarrow A$ | $A_{n+1} \subseteq A_n$ for all $n$ and $A = \bigcap_{n=1}^\infty A_n$ |
| $\mathbb{E}[X]$ | Expectation of a random variable $X$ whenever $X$ is integrable |
| $\mathbb{I}_A$ | Indicator function of the event $A$ |

## 3. Pi-systems and Lambda-systems

<div class="math-box box-definition">
<span class="math-box-title">Definition 3.1 — $\pi$-system</span>

A nonempty collection $\mathcal{P}$ of subsets of a nonempty set $\Omega$ is a **$\pi$-system** if it is closed under finite intersections. In particular,

$$A, B \in \mathcal{P} \implies A \cap B \in \mathcal{P}.$$

</div>

<div class="math-box box-definition">
<span class="math-box-title">Definition 3.2 — $\lambda$-system</span>

A collection $\mathcal{L}$ of subsets of a nonempty set $\Omega$ is a **$\lambda$-system** if it satisfies:

1. $\Omega \in \mathcal{L}$,
2. $A \in \mathcal{L} \implies A^c \in \mathcal{L}$,
3. if $(A_i)_{i=1}^\infty \subseteq \mathcal{L}$ are pairwise disjoint (i.e. $A_i \cap A_j = \varnothing$ for $i \neq j$), then $\displaystyle\bigcup_{i=1}^\infty A_i \in \mathcal{L}$.

</div>

<div class="math-box box-theorem">
<span class="math-box-title">Theorem 3.3 — $\pi$-system $\cap$ $\lambda$-system $\Longleftrightarrow$ $\sigma$-field</span>

A collection $\mathcal{F}$ of subsets of a nonempty set $\Omega$ is a $\pi$-system and a $\lambda$-system if and only if $\mathcal{F}$ is a $\sigma$-field.

</div>

<div class="math-proof" markdown="1">

**Proof.**

**(⇐)** If $\mathcal{F}$ is a $\sigma$-field, then it is closed under intersections and satisfies (i)–(iii), hence it is both a $\pi$-system and a $\lambda$-system.

**(⇒)** Assume $\mathcal{F}$ is a $\pi$-system and a $\lambda$-system. To show that $\mathcal{F}$ is a $\sigma$-field, it remains to show closure under countable unions for an arbitrary sequence $(A_i)_{i=1}^\infty \subseteq \mathcal{F}$. Define

$$B_1 := A_1, \qquad B_i := A_i \setminus \bigcup_{j=1}^{i-1} A_j \quad (i \ge 2).$$

Then $B_i \in \mathcal{F}$ for all $i$ and $(B_i)_{i=1}^\infty$ are pairwise disjoint. Moreover,

$$\bigcup_{i=1}^\infty A_i = \bigcup_{i=1}^\infty B_i.$$

Since $\mathcal{F}$ is a $\lambda$-system, $\bigcup_{i=1}^\infty B_i \in \mathcal{F}$, hence $\bigcup_{i=1}^\infty A_i \in \mathcal{F}$. Therefore $\mathcal{F}$ is a $\sigma$-field. $\blacksquare$

</div>

<div class="math-box box-lemma">
<span class="math-box-title">Lemma 3.4 — $\lambda$-system generated by an element</span>

Let $\mathcal{L}$ be a $\lambda$-system. For $A \in \mathcal{L}$, define

$$\mathcal{G}_A := \{B \subseteq \Omega : A \cap B \in \mathcal{L}\}.$$

Then $\mathcal{G}_A$ is also a $\lambda$-system for every $A \in \mathcal{L}$.

</div>

<div class="math-proof" markdown="1">

**Proof.** We verify the following three properties.

**(i)** $\Omega \in \mathcal{G}_A$: We have $A \cap \Omega = A \in \mathcal{L}$.

**(ii)** Closed under complements: Let $B \in \mathcal{G}_A$, i.e. $A \cap B \in \mathcal{L}$. Note that

$$B^c \cap A \in \mathcal{L} \iff B \cup A^c \in \mathcal{L} \iff (B \cap A) \cup A^c \in \mathcal{L}.$$

Also note that $\mathcal{L}$ is closed under finite disjoint unions, which proves the claim.

**(iii)** Closed under countable disjoint unions: Let $(B_n)_{n=1}^\infty \subseteq \mathcal{G}_A$ be pairwise disjoint. Then $A \cap B_n \in \mathcal{L}$ and $(A \cap B_n)$ are pairwise disjoint. By the $\lambda$-system property,

$$A \cap \left(\bigcup_{n=1}^\infty B_n\right) = \bigcup_{n=1}^\infty (A \cap B_n) \in \mathcal{L}.$$

Hence $\bigcup_{n=1}^\infty B_n \in \mathcal{G}_A$. $\blacksquare$

</div>

<div class="math-box box-theorem">
<span class="math-box-title">Theorem 3.5 — Dynkin's $\pi$-$\lambda$ Theorem</span>

Let $\mathcal{P}$ be a $\pi$-system. If $\mathcal{P} \subseteq \mathcal{L}$ where $\mathcal{L}$ is a $\lambda$-system, then $\sigma(\mathcal{P}) \subseteq \mathcal{L}$.

</div>

<div class="math-proof" markdown="1">

**Proof.** Let $\lambda(\mathcal{P})$ be the smallest $\lambda$-system containing $\mathcal{P}$. For any $A \in \mathcal{P}$ and $B \in \mathcal{P}$, since $\mathcal{P}$ is a $\pi$-system, $A \cap B \in \mathcal{P} \subseteq \lambda(\mathcal{P})$, so $B \in \mathcal{G}_A$.

Therefore $\mathcal{P} \subseteq \mathcal{G}_A$ for all $A \in \mathcal{P}$, and since $\mathcal{G}_A$ is a $\lambda$-system (by Lemma 3.4), $\lambda(\mathcal{P}) \subseteq \mathcal{G}_A$ for all $A \in \mathcal{P}$.

This means for any $B \in \lambda(\mathcal{P})$ and $A \in \mathcal{P}$, $A \cap B \in \lambda(\mathcal{P})$, i.e. $A \in \mathcal{G}_B$. Hence $\mathcal{P} \subseteq \mathcal{G}_B$ for all $B \in \lambda(\mathcal{P})$, and again $\lambda(\mathcal{P}) \subseteq \mathcal{G}_B$ for all $B \in \lambda(\mathcal{P})$.

Therefore $\lambda(\mathcal{P})$ is closed under intersections, so it is a $\pi$-system. By Theorem 3.3, $\lambda(\mathcal{P})$ is a $\sigma$-field. Since $\sigma(\mathcal{P}) \subseteq \lambda(\mathcal{P}) \subseteq \mathcal{L}$, we conclude $\sigma(\mathcal{P}) \subseteq \mathcal{L}$. $\blacksquare$

</div>
