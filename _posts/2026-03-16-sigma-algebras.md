## Notation

| Symbol | Meaning |
|--------|---------|
| $(\Omega, \mathcal{F}, \mathbb{P})$ | A probability space, where $\Omega$ is the sample space, $\mathcal{F}$ is a $\sigma$-field on $\Omega$, and $\mathbb{P}$ is a probability measure on $\mathcal{F}$ |
| $\mathcal{P}(\Omega)$ | The power set of $\Omega$ |
| $A^c$ | The complement of $A$ in $\Omega$ |
| $\sigma(\mathcal{G})$ | The $\sigma$-field generated by $\mathcal{G}$ |
| $\mathcal{B}(\mathbb{R})$ | The Borel $\sigma$-field on $\mathbb{R}$ |
| $A_n \uparrow A$ | $A_n \subseteq A_{n+1}$ for all $n$ and $A = \bigcup_{n=1}^\infty A_n$ |
| $A_n \downarrow A$ | $A_{n+1} \subseteq A_n$ for all $n$ and $A = \bigcap_{n=1}^\infty A_n$ |
| $\mathbb{E}[X]$ | Expectation of a random variable $X$ whenever $X$ is integrable |
| $\mathbb{I}_A$ | Indicator function of the event $A$ |

## 1. Sigma-Algebras

<div class="math-box box-definition">
<span class="math-box-title">Definition 1.1 — $\sigma$-field</span>

Let $\Omega$ be a nonempty set. A family $\mathcal{F} \subseteq 2^\Omega$ is a **$\sigma$-field** (also called a **$\sigma$-algebra**) if:

1. $\Omega \in \mathcal{F}$,
2. $A \in \mathcal{F} \implies A^c \in \mathcal{F}$,
3. if $A_n \in \mathcal{F}$ for $n = 1, 2, \dots$, then $\displaystyle\bigcup_{n=1}^\infty A_n \in \mathcal{F}$.

</div>

<div class="math-box box-example">
<span class="math-box-title">Example 1.2 — Trivial $\sigma$-fields</span>

Two extreme examples are:

- **(a)** $\mathcal{F} = \{\varnothing, \Omega\}$, the **smallest** $\sigma$-field on $\Omega$,
- **(b)** $\mathcal{F} = \mathcal{P}(\Omega)$, the **largest** $\sigma$-field on $\Omega$.

</div>

<div class="math-box box-definition">
<span class="math-box-title">Definition 1.3 — Generated $\sigma$-field</span>

Let $\Omega$ be a nonempty set and let $\mathcal{G} \subseteq 2^\Omega$ be any family of subsets. The **$\sigma$-field generated by $\mathcal{G}$**, denoted $\sigma(\mathcal{G})$, is the smallest $\sigma$-field containing $\mathcal{G}$, i.e.

$$\sigma(\mathcal{G}) := \bigcap\{\mathcal{F} \subseteq 2^\Omega : \mathcal{F} \text{ is a } \sigma\text{-field and } \mathcal{G} \subseteq \mathcal{F}\}.$$

Equivalently, $\sigma(\mathcal{G})$ is a $\sigma$-field with $\mathcal{G} \subseteq \sigma(\mathcal{G})$, and for any $\sigma$-field $\mathcal{F}$ with $\mathcal{G} \subseteq \mathcal{F}$ we have $\sigma(\mathcal{G}) \subseteq \mathcal{F}$.

</div>

<div class="math-box box-example">
<span class="math-box-title">Example 1.4 — Borel $\sigma$-field on $\mathbb{R}$</span>

Let $\Omega = \mathbb{R}$ and let

$$\mathcal{G} := \{(a,b] : a \le b,\ a,b \in \mathbb{R}\}.$$

Then $\sigma(\mathcal{G})$ is the Borel $\sigma$-field $\mathcal{B}(\mathbb{R})$.

</div>

<div class="math-box box-definition">
<span class="math-box-title">Definition 1.5 — Borel $\sigma$-field on $\mathbb{R}$ via topology</span>

Let $\mathcal{T}$ be the usual topology on $\mathbb{R}$, i.e.

$$\mathcal{T} := \{A \subseteq \mathbb{R} : A \text{ is open}\}.$$

The **Borel $\sigma$-field** on $\mathbb{R}$ is defined by

$$\mathcal{B}(\mathbb{R}) := \sigma(\mathcal{T}).$$

</div>

<div class="math-box box-remark">
<span class="math-box-title">Remark 1.6 — Open and closed sets</span>

A set $A \subseteq \mathbb{R}$ is open if and only if for every $x \in A$ there exists $a_x < b_x$ such that $(a_x, b_x) \subseteq A$. A set $A$ is closed if and only if $A^c$ is open.

</div>

<div class="math-box box-theorem">
<span class="math-box-title">Theorem 1.7 — $\mathcal{B}(\mathbb{R})$ is generated by rational open intervals</span>

Let

$$\mathcal{I} := \{(a,b) : a < b,\ a,b \in \mathbb{Q}\}.$$

Then $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$.

</div>

<div class="math-proof">

**Proof.** First, $\mathcal{I} \subseteq \mathcal{T}$ since every $(a,b)$ is open. Hence $\sigma(\mathcal{I}) \subseteq \sigma(\mathcal{T}) = \mathcal{B}(\mathbb{R})$.

For the reverse inclusion, let $A \in \mathcal{T}$ be open. For each $x \in A$, choose $a_x, b_x$ with $(a_x, b_x) \subseteq A$. Pick rationals $p_x < q_x \in \mathbb{Q}$ such that

$$x \in (p_x, q_x) \subseteq (a_x, b_x) \subseteq A.$$

Then $A = \bigcup_{x \in A}(p_x, q_x)$. Since $\{(p_x, q_x) : x \in A\} \subseteq \mathcal{I}$ and $\mathcal{I}$ is countable, $A \in \sigma(\mathcal{I})$. Therefore $\mathcal{T} \subseteq \sigma(\mathcal{I})$, so $\mathcal{B}(\mathbb{R}) = \sigma(\mathcal{T}) \subseteq \sigma(\mathcal{I})$. Combining both inclusions yields $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$. $\blacksquare$

</div>

<div class="math-box box-definition">
<span class="math-box-title">Definition 1.8 — Interval classes on $\mathbb{R}$</span>

Define

$$\mathcal{I}' := \{(a,b) : -\infty < a < b < \infty,\ a,b \in \mathbb{R}\}, \qquad \mathcal{I}'' := \{[a,b] : -\infty < a < b < \infty,\ a,b \in \mathbb{R}\}.$$

</div>

<div class="math-box box-theorem">
<span class="math-box-title">Theorem 1.9 — $\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R})$</span>

$$\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R}).$$

</div>

<div class="math-proof">

**Proof.** Since $\mathcal{I} \subseteq \mathcal{I}' \subseteq \mathcal{T}$, we have $\sigma(\mathcal{I}) \subseteq \sigma(\mathcal{I}') \subseteq \sigma(\mathcal{T}) = \mathcal{B}(\mathbb{R})$. By Theorem 1.7, $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$. Hence $\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R})$. $\blacksquare$

</div>

<div class="math-box box-theorem">
<span class="math-box-title">Theorem 1.10 — $\sigma(\mathcal{I}') = \sigma(\mathcal{I}'')$</span>

$$\sigma(\mathcal{I}') = \sigma(\mathcal{I}'').$$

</div>

<div class="math-proof">

**Proof.**

**(⊆)** For any $(a,b) \in \mathcal{I}'$,

$$(a,b) = \bigcup_{n=1}^{\infty}\left[a + \tfrac{1}{n},\ b - \tfrac{1}{n}\right].$$

Hence $(a,b) \in \sigma(\mathcal{I}'')$, so $\sigma(\mathcal{I}') \subseteq \sigma(\mathcal{I}'')$.

**(⊇)** For any $[a,b] \in \mathcal{I}''$,

$$[a,b] = \bigcap_{n=1}^{\infty}\left(a - \tfrac{1}{n},\ b + \tfrac{1}{n}\right).$$

Hence $[a,b] \in \sigma(\mathcal{I}')$, so $\sigma(\mathcal{I}'') \subseteq \sigma(\mathcal{I}')$. $\blacksquare$

</div>
