## Notation

| Symbol | Meaning |
|--------|---------|
| $(\Omega, \mathcal{F}, \mathbb{P})$ | A probability space, where $\Omega$ is the sample space, $\mathcal{F}$ is a $\sigma$-field on $\Omega$, and $\mathbb{P}$ is a probability measure on $\mathcal{F}$ |
| $\mathcal{P}(\Omega)$ | The power set of $\Omega$ |
| $A^c$ | The complement of $A$ in $\Omega$ |
| $\sigma(\mathcal{G})$ | The $\sigma$-field generated by $\mathcal{G}$ |
| $\mathcal{B}(\mathbb{R})$ | The Borel $\sigma$-field on $\mathbb{R}$ |
| $A_n \uparrow A$ | $A_n \subseteq A_{n+1}$ for all $n$ and $A = \bigcup_{n=1}^\infty A_n$ |
| $A_n \downarrow A$ | $A_{n+1} \subseteq A_n$ for all $n$ and $A = \bigcap_{n=1}^\infty A_n$ |
| $\mathbb{E}[X]$ | Expectation of a random variable $X$ whenever $X$ is integrable |
| $\mathbb{I}_A$ | Indicator function of the event $A$ |

## 1. Sigma-Algebras

### Definition 1.1 — $\sigma$-field

Let $\Omega$ be a nonempty set. A family $\mathcal{F} \subseteq 2^\Omega$ is a **$\sigma$-field** (also called a **$\sigma$-algebra**) if

1. $\Omega \in \mathcal{F}$,
2. $A \in \mathcal{F} \implies A^c \in \mathcal{F}$,
3. if $A_n \in \mathcal{F}$ for $n = 1, 2, \dots$, then $\displaystyle\bigcup_{n=1}^\infty A_n \in \mathcal{F}$.

---

### Example 1.2 — Trivial $\sigma$-fields

Two extreme examples are:

- **(a)** $\mathcal{F} = \{\varnothing, \Omega\}$, the **smallest** $\sigma$-field on $\Omega$,
- **(b)** $\mathcal{F} = \mathcal{P}(\Omega)$, the **largest** $\sigma$-field on $\Omega$.

---

### Definition 1.3 — Generated $\sigma$-field

Let $\Omega$ be a nonempty set and let $\mathcal{G} \subseteq 2^\Omega$ be any family of subsets. The **$\sigma$-field generated by $\mathcal{G}$**, denoted $\sigma(\mathcal{G})$, is the smallest $\sigma$-field containing $\mathcal{G}$, i.e.

$$
\sigma(\mathcal{G}) := \bigcap\bigl\{\mathcal{F} \subseteq 2^\Omega : \mathcal{F} \text{ is a } \sigma\text{-field and } \mathcal{G} \subseteq \mathcal{F}\bigr\}.
$$

Equivalently, $\sigma(\mathcal{G})$ is a $\sigma$-field with $\mathcal{G} \subseteq \sigma(\mathcal{G})$, and for any $\sigma$-field $\mathcal{F}$ with $\mathcal{G} \subseteq \mathcal{F}$ we have $\sigma(\mathcal{G}) \subseteq \mathcal{F}$.

---

### Example 1.4 — Borel $\sigma$-field on $\mathbb{R}$

Let $\Omega = \mathbb{R}$ and let

$$
\mathcal{G} := \{(a,b] : a \le b,\ a,b \in \mathbb{R}\}.
$$

Then $\sigma(\mathcal{G})$ is the Borel $\sigma$-field $\mathcal{B}(\mathbb{R})$.

---

### Definition 1.5 — Borel $\sigma$-field on $\mathbb{R}$ via topology

Let $\mathcal{T}$ be the usual topology on $\mathbb{R}$, i.e.

$$
\mathcal{T} := \{A \subseteq \mathbb{R} : A \text{ is open}\}.
$$

The **Borel $\sigma$-field** on $\mathbb{R}$ is defined by

$$
\mathcal{B}(\mathbb{R}) := \sigma(\mathcal{T}).
$$

---

### Remark 1.6 — Open and closed sets

A set $A \subseteq \mathbb{R}$ is open if and only if for every $x \in A$ there exists $a_x < b_x$ such that $(a_x, b_x) \subseteq A$. A set $A$ is closed if and only if $A^c$ is open.

---

### Theorem 1.7 — $\mathcal{B}(\mathbb{R})$ is generated by rational open intervals

Let

$$
\mathcal{I} := \{(a,b) : a < b,\ a,b \in \mathbb{Q}\}.
$$

Then $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$.

**Proof.**
First, $\mathcal{I} \subseteq \mathcal{T}$ since every $(a,b)$ is open. Hence $\sigma(\mathcal{I}) \subseteq \sigma(\mathcal{T}) = \mathcal{B}(\mathbb{R})$.

For the reverse inclusion, let $A \in \mathcal{T}$ be open. For each $x \in A$, choose $a_x, b_x$ with $(a_x, b_x) \subseteq A$. Pick rationals $p_x < q_x \in \mathbb{Q}$ such that

$$
x \in (p_x, q_x) \subseteq (a_x, b_x) \subseteq A.
$$

Then $A = \bigcup_{x \in A}(p_x, q_x)$. Since $\{(p_x, q_x) : x \in A\} \subseteq \mathcal{I}$ and $\mathcal{I}$ is countable, $A \in \sigma(\mathcal{I})$. Therefore $\mathcal{T} \subseteq \sigma(\mathcal{I})$, so $\mathcal{B}(\mathbb{R}) = \sigma(\mathcal{T}) \subseteq \sigma(\mathcal{I})$.

Combining both inclusions yields $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$. $\blacksquare$

---

### Definition 1.8 — Interval classes on $\mathbb{R}$

Define

$$
\mathcal{I}' := \{(a,b) : -\infty < a < b < \infty,\ a,b \in \mathbb{R}\}, \qquad \mathcal{I}'' := \{[a,b] : -\infty < a < b < \infty,\ a,b \in \mathbb{R}\}.
$$

---

### Theorem 1.9 — $\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R})$

$$\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R}).$$

**Proof.**
Since $\mathcal{I} \subseteq \mathcal{I}' \subseteq \mathcal{T}$, we have

$$
\sigma(\mathcal{I}) \subseteq \sigma(\mathcal{I}') \subseteq \sigma(\mathcal{T}) = \mathcal{B}(\mathbb{R}).
$$

By Theorem 1.7, $\sigma(\mathcal{I}) = \mathcal{B}(\mathbb{R})$. Hence $\sigma(\mathcal{I}') = \mathcal{B}(\mathbb{R})$. $\blacksquare$

---

### Theorem 1.10 — $\sigma(\mathcal{I}') = \sigma(\mathcal{I}'')$

$$\sigma(\mathcal{I}') = \sigma(\mathcal{I}'').$$

**Proof.**
**(⊆)** For any $(a,b) \in \mathcal{I}'$,

$$
(a,b) = \bigcup_{n=1}^{\infty}\left[a + \tfrac{1}{n},\ b - \tfrac{1}{n}\right].
$$

Hence $(a,b) \in \sigma(\mathcal{I}'')$, so $\mathcal{I}' \subseteq \sigma(\mathcal{I}'')$, and therefore $\sigma(\mathcal{I}') \subseteq \sigma(\mathcal{I}'')$.

**(⊇)** For any $[a,b] \in \mathcal{I}''$,

$$
[a,b] = \bigcap_{n=1}^{\infty}\left(a - \tfrac{1}{n},\ b + \tfrac{1}{n}\right).
$$

Hence $[a,b] \in \sigma(\mathcal{I}')$, so $\mathcal{I}'' \subseteq \sigma(\mathcal{I}')$, and therefore $\sigma(\mathcal{I}'') \subseteq \sigma(\mathcal{I}')$. $\blacksquare$

## 2. Probability Spaces

### Definition 2.1 — Measure

Let $(\Omega, \mathcal{F})$ be a measurable space. A set function

$$
\mu : \mathcal{F} \to [0, \infty]
$$

is called a **measure** if

1. $\mu(\varnothing) = 0$,
2. for any pairwise disjoint sequence $(A_i)_{i=1}^\infty \subseteq \mathcal{F}$ (i.e. $A_i \cap A_j = \varnothing$ for $i \neq j$),

$$
\mu\!\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} \mu(A_i).
$$

---

### Remark 2.2 — Probability measure

If $\mu(\Omega) = 1$, we say that $\mu$ is a **probability measure** and denote it by $\mathbb{P}$.

---

### Definition 2.3 — Probability space

A **probability space** is a triple $(\Omega, \mathcal{F}, \mathbb{P})$, where $(\Omega, \mathcal{F})$ is a measurable space and $\mathbb{P}$ is a probability measure on $\mathcal{F}$.

---

### Theorem 2.4 — Properties of a probability measure

Let $(\Omega, \mathcal{F}, \mathbb{P})$ be a probability space. Then:

- **(a) Monotonicity:** if $A \subseteq B$, then $\mathbb{P}(A) \le \mathbb{P}(B)$.
- **(b) Continuity from below:** if $A_n \uparrow A$ (i.e. $A_n \subseteq A_{n+1}$ and $A = \bigcup_{n=1}^\infty A_n$), then

$$
\lim_{n \to \infty} \mathbb{P}(A_n) = \mathbb{P}(A).
$$

**Proof.**
**(a)** Note first that countable additivity implies finite additivity for mutually exclusive events: if $A_1, \dots, A_m \in \mathcal{F}$ are pairwise disjoint, then

$$
\mathbb{P}\!\left(\bigcup_{i=1}^{m} A_i\right) = \sum_{i=1}^{m} \mathbb{P}(A_i).
$$

Now assume $A \subseteq B$. Then $B = A \cup (B \setminus A)$ and $A \cap (B \setminus A) = \varnothing$, hence

$$
\mathbb{P}(B) = \mathbb{P}(A) + \mathbb{P}(B \setminus A) \ge \mathbb{P}(A).
$$

**(b)** Assume $A_n \uparrow A$, so $A = \bigcup_{n=1}^\infty A_n$. Define

$$
B_1 := A_1, \qquad B_n := A_n \setminus \bigcup_{j=1}^{n-1} A_j \quad (n \ge 2).
$$

Then $B_n \in \mathcal{F}$ are pairwise disjoint and $\bigcup_{n=1}^\infty B_n = \bigcup_{n=1}^\infty A_n = A$. Also, for each $n$,

$$
A_n = \bigcup_{k=1}^{n} B_k.
$$

Therefore,

$$
\mathbb{P}(A) = \mathbb{P}\!\left(\bigcup_{k=1}^\infty B_k\right) = \sum_{k=1}^\infty \mathbb{P}(B_k), \qquad \mathbb{P}(A_n) = \mathbb{P}\!\left(\bigcup_{k=1}^{n} B_k\right) = \sum_{k=1}^{n} \mathbb{P}(B_k).
$$

Taking $n \to \infty$ gives

$$
\lim_{n \to \infty} \mathbb{P}(A_n) = \lim_{n \to \infty} \sum_{k=1}^{n} \mathbb{P}(B_k) = \sum_{k=1}^\infty \mathbb{P}(B_k) = \mathbb{P}(A). \quad \blacksquare
$$

---

### Remark 2.5 — Continuity from above

If $A_n \downarrow A$, then

$$
\lim_{n \to \infty} \mathbb{P}(A_n) = \mathbb{P}(A).
$$

## 3. Pi-systems and Lambda-systems

### Definition 3.1 — $\pi$-system

A nonempty collection $\mathcal{P}$ of subsets of a nonempty set $\Omega$ is a **$\pi$-system** if it is closed under finite intersections. In particular,

$$
A, B \in \mathcal{P} \implies A \cap B \in \mathcal{P}.
$$

---

### Definition 3.2 — $\lambda$-system

A collection $\mathcal{L}$ of subsets of a nonempty set $\Omega$ is a **$\lambda$-system** if it satisfies:

1. $\Omega \in \mathcal{L}$,
2. $A \in \mathcal{L} \implies A^c \in \mathcal{L}$,
3. if $(A_i)_{i=1}^\infty \subseteq \mathcal{L}$ are pairwise disjoint (i.e. $A_i \cap A_j = \varnothing$ for $i \neq j$), then

$$
\bigcup_{i=1}^\infty A_i \in \mathcal{L}.
$$

---

### Theorem 3.3 — $\mathcal{F}$ is a $\pi$-system and a $\lambda$-system $\Longleftrightarrow$ $\mathcal{F}$ is a $\sigma$-field

A collection $\mathcal{F}$ of subsets of a nonempty set $\Omega$ is a $\pi$-system and a $\lambda$-system if and only if $\mathcal{F}$ is a $\sigma$-field.

**Proof.**
**(⇐)** If $\mathcal{F}$ is a $\sigma$-field, then it is closed under intersections and satisfies (i)–(iii), hence it is both a $\pi$-system and a $\lambda$-system.

**(⇒)** Assume $\mathcal{F}$ is a $\pi$-system and a $\lambda$-system. To show that $\mathcal{F}$ is a $\sigma$-field, it remains to show closure under countable unions for an arbitrary sequence $(A_i)_{i=1}^\infty \subseteq \mathcal{F}$.

Define

$$
B_1 := A_1, \qquad B_i := A_i \setminus \bigcup_{j=1}^{i-1} A_j \quad (i \ge 2).
$$

Then $B_i \in \mathcal{F}$ for all $i$ and $(B_i)_{i=1}^\infty$ are pairwise disjoint. Moreover,

$$
\bigcup_{i=1}^\infty A_i = \bigcup_{i=1}^\infty B_i.
$$

Since $\mathcal{F}$ is a $\lambda$-system, $\bigcup_{i=1}^\infty B_i \in \mathcal{F}$, hence $\bigcup_{i=1}^\infty A_i \in \mathcal{F}$. Therefore $\mathcal{F}$ is a $\sigma$-field. $\blacksquare$
